# Install unity-game-dev for Gemini CLI

## Preferred

```bash
npx skills add gigio1023/unity-game-dev-skill@unity-game-dev --agent gemini-cli
```

## Manual install

```bash
git clone https://github.com/gigio1023/unity-game-dev-skill.git ~/.gemini/unity-game-dev-skill
mkdir -p ~/.gemini/skills/unity-game-dev
cp -R ~/.gemini/unity-game-dev-skill/. ~/.gemini/skills/unity-game-dev/
```

Restart Gemini CLI after copying.
