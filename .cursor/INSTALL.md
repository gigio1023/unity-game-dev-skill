# Install unity-game-dev for Cursor

## Preferred

```bash
npx skills add gigio1023/unity-game-dev-skill@unity-game-dev --agent cursor
```

## Manual install

```bash
git clone https://github.com/gigio1023/unity-game-dev-skill.git ~/.cursor/unity-game-dev-skill
mkdir -p ~/.cursor/skills/unity-game-dev
cp -R ~/.cursor/unity-game-dev-skill/. ~/.cursor/skills/unity-game-dev/
```

Restart Cursor after copying.
