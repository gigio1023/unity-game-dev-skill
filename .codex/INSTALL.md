# Install unity-game-dev for Codex

## Preferred

```bash
npx skills add gigio1023/unity-game-dev-skill@unity-game-dev
```

## Manual install

```bash
git clone https://github.com/gigio1023/unity-game-dev-skill.git ~/.codex/unity-game-dev-skill
mkdir -p ~/.codex/skills/unity-game-dev
cp -R ~/.codex/unity-game-dev-skill/. ~/.codex/skills/unity-game-dev/
```

Restart Codex after copying.
